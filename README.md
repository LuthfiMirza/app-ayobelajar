# Ayo Belajar Platform

Platform pendidikan AI-powered untuk daerah 3T Indonesia.

## Features

- ?? AI ChatBot dengan Azure OpenAI
- ?? Multi-language translator dengan bahasa daerah
- ?? Sistem manajemen modul pembelajaran
- ?? User management dan authentication
- ?? Responsive design untuk mobile

## Technology Stack

- **Backend**: Laravel 11 (PHP 8.2)
- **Frontend**: Blade Templates, Livewire, TailwindCSS
- **Database**: MySQL
- **Storage**: Azure Blob Storage
- **AI Services**: Azure OpenAI, Azure Translator
- **Hosting**: Azure App Service
